Lists

Use lists to organize groups of similar data.

There are different list types, and each is useful in its own way.

  • Standard Lists (bullets or numbers) are ubiquitous.
  • Description Lists (a.k.a. 'definition lists') are well suited to dictionaries / glossaries or brief FAQs.
  • List Groups are well-suited to clearly delineating lists visually, and are strongly complemented by badges.

Custom CSS

For Description Lists, you may wish to add some margin-bottom to each description pair.

For List Groups, you may wish to set the <a> element to display: block so that users may activate the link from anywhere inside the list element.

You may add the .active class to any <li> or <button> tag to highlight an active row.

Get all bootstrapr CSS 
/* Add bottom margin to the description lists */
dd {
   margin-bottom: 1rem;
}

/* Set the anchor element to display block */
.list-group li a {
   display: block;
}
