Blockquotes

Blockquotes are brief, impactful textual excerpts.

Blockquotes are especially useful as epigraphs introducing content by highlighting its main ideas or themes.

Example | And when this happens...
Example | Rev. Dr. Martin Luther King, Jr.
Example | "I Have a Dream"
Example | http://www.link.com

And when this happens, when we allow freedom to ring, when we let it ring from every village and every hamlet, from every state and every city, we will be able to speed up that day when all of God's children, black men and white men, Jews and Gentiles, Protestants and Catholics, will be able to join hands and sing in the words of the old Negro spiritual: “Free at last! Free at last! Thank God Almighty, we are free at last!”

Rev. Martin Luther King, Jr., from “I Have a Dream”

Custom CSS

Custom CSS may be required for adding links to the <cite> element in order to keep the <a> element the same size as its containing element.

/* Maintain the same size for the anchor element (link) as the cite element */
cite a {
   font-size: inherit;
}
