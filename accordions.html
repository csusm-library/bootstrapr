Custom CSS

Custom CSS (and Font Awesome 4) is required for the light blue plus symbols on the panels.

/* Adds 'plus' glyph for enabling collapsible panels */
.panel-heading .accordion-toggle:before {
   font-family: FontAwesome;
   content: "\f068";
   float: right;
   color: lightsteelblue;
}

.panel-heading .accordion-toggle.collapsed:before {
   content: "\f067";
   color: lightsteelblue;
}

/* sets the link to the size of the entire panel title */
.panel-title > a {
   display: block;
}

Notes

Keep Accordion Sets Unique
While you may place multiple accordion sets on a page, do not simply cut and paste the same set again and again. Make sure to generate each new accordion set anew, ensuring the random number assigned to each set stays unique and does not conlflict with the other sets on the page.
