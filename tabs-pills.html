Tabs & Pills

Use tabs and pills to conceal-and-reveal closely related content.

Tabs should be limited in number and each tab should have a short, simple title. As a rule, if you are building tabs across multiple rows, you may need to reexamine the information architecture for your site.

Custom CSS

To add padding to the tab panels, you may wish to specify padding for the .tab-pane class.

Get all bootstrapr CSS 
/* Adds padding around the content inside each tab panel */
.tab-pane {
   padding: 1rem;
}

Notes

Disabling a Tab
Disable a tab by 1) adding the .disabled class and 2) removing the data-toggle='tab' declaration on the <li> element

Ã— Keep Tab Sets Unique
While you may place multiple tab sets on a page...do not simply cut and paste the same set again and again. Make sure to generate each new tab set anew. This will ensure the random number assigned to each set stays unique and does not conlflict with the other sets on the page.
