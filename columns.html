Bootstrap 3.3.7 | Columns

Columns

Use columns to structure your page content.

Although many more column combinations are possible, I have included the most common column layouts here.

Columns (e.g., <div class="col-sm-*">) live inside rows (<div class="row">). By default we have placed each set of columns inside a single row. However, you may also place an unlimited number of columns inside a single row (see the example code), and as column widths surpass 12 columns, the next columns are pushed down to the next line.

 Instructions
Select Column sets to add them to the stack. Click + Drag to reorder them. Click a column set to delete it from the stack. Click Build to generate the code.
