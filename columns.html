Columns

Use columns to structure your page content.

Although many more column combinations are possible, I have included the most common column layouts here. Columns live inside <rows> and by default we have placed each column selection inside a single row.

However, you may place an unlimited number of columns inside a single row (see the example code). As column widths surpass 12 columns, the next columns are pushed down to the next line.
