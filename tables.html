Tables

Use tables to display data you wish to compare and contrast.

You may style your tables with any combination of the attributes below. Use tables only for tabular data, not as faux columns. If you need them, use columns to divide your page, instead.

content content content
content content content
content content content 
<table class='table table-striped'>
   <thead>
      <tr>
         <th>content</th>
         <th>content</th>
         <th>content</th>
      </tr>
   </thead>
   <tbody>
      <tr>
         <td>content</td>
         <td>content</td>
         <td>content</td>
      </tr>
      <tr>
         <td>content</td>
         <td>content</td>
         <td>content</td>
      </tr>
   </tbody>
</table>

Notes

Accent Your Data With Colors! Change the background color of any table row (or table data) element by adding the following classes:

grey .active <tr class="active">content<tr>
blue .info <tr class="info">content<tr>
green .success <tr class="success">content<tr>
yellow .warning <tr class="warning">content<tr>
red .danger <tr class="danger">content<tr>

Custom CSS

Custom CSS is required for the steps column and for removing borders from tables.

/* Set styles for the Steps column */
.numbered-steps tr td:first-of-type {
   padding: 3px;
   text-align: center;
   vertical-align: middle;
   font-weight: 600;
   background: aliceblue;
   width: 1.5rem;
   color: #666666;
   font-size: 1.2rem;
}

/* Set the font color to gray */
.gray {
   color: gray;
}

/* Remove all borders from table */
table.table-borderless tr,
table.table-borderless th,
table.table-borderless td {
   border: none !important;
}
