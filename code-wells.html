Code Wells

Code wells display computer language in a highly readable format.

Code Wells are rendered with the <pre> element. Google's Prettify package is used to color-code the output. Unlike other HTML elements, the <pre> element preserves all spacing exactly as you type it.

// This is a javascript comment
$( document ).ready(function() {
   $('.navfeatures').addClass('active');
   $('.navcodewells').addClass('active');
});

<pre class='prettyprint'>
// This is a javascript comment
$( document ).ready(function() {
   $('.navfeatures').addClass('active');
   $('.navcodewells').addClass('active');
});
</pre>

Custom CSS

Custom CSS is required if you wish to disable the alternating backgrounds on numbered lines. Alternating backgrounds are fine on lighter backgrounds but generally look poor on darker backgrounds.

/* Disables the background for alternating line numbers */
ol.linenums li {
   background: transparent !important;
}
