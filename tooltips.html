Tooltips

Tooltips provide contextual information on mouse-over.

To add a tooltip to any element, add the attributes of data-toggle="tooltip", data-placement="[top|bottom|left|right]", and title="Tooltip content here".

You will also need to add the jQuery tooltip() method (see below) to your page to activate the tooltips.

Custom CSS

Custom CSS is required to change the tooltip background, font color, font size, margin and padding of your tooltips.

View all Bootstrapr CSS 
.tooltip .tooltip-inner {
  background-color: darkred ;
  color: white;
  font-size: 1.25rem;
  margin-top: 10px;
  padding: 8px;
  }

Custom JS

Custom javascript may be required to activate the tooltips in the page.

$('[data-toggle="tooltip"]').tooltip();
