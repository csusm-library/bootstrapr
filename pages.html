Pages

All bootstrap pages are built on a 12-column grid.

Fill in the fields below to customize a skeleton Bootstrap page. Then add Bootstrap components from the Features and Containers tabs to the main content div—the one with the id="wrapper".

Ã— Required Libraries
Bootstrap 3 requires the jQuery libraries. If you choose to serve Bootstrap from a CDN, links will be included in the generated HTML. If you choose to serve the files locally, you will need to download jQuery and download Bootstrap and place them in your project directory. Note that both Bootstrap and jQuery are already included for the LibGuides platform.
